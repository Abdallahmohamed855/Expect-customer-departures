Learn a model that was developed to predict, including whether a bank customer will continue with the bank or leave (customer forecasting), by analyzing customer data such as age, customer, number of banking products, duration of dealing with the bank, banking activity, and income. It aims to assist the bank in dealing with customers to immediately prepare for pre-emptive measures for air attacks on them. Data processing, selection, data harvesting, and thus classification techniques such as Logistic Regression, Random Forest, and XGBoost have been used to achieve the highest achievements in history. This model provides significant value to the bank through improved member growth, expansion and customer reduction
